## URL

## Setup & Local Dev
- Get Gatsby: `npm install -g gatsby-cli`

- `git clone git@github.com:sreejagpt/gatsby-blog.git`

- `cd gatsby-blog`

- `gatsby develop`

## Build

`gatsby build`
